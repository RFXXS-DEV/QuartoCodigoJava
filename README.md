# Java: Consumindo API, Gravando Arquivos e Lidando com Erros

Repositório criado para armazenar os exercícios, desafios e projetos desenvolvidos durante o curso **Java: Consumindo API, Gravando Arquivos e Lidando com Erros**, disponibilizado pelo programa **Geração Caldeira**.

## Conteúdos estudados

- Consumo de APIs em Java
- Requisições HTTP
- Manipulação de dados recebidos de APIs
- Tratamento de exceções
- Boas práticas para lidar com erros
- Leitura e gravação de arquivos
- Persistência de dados em arquivos
- Organização de código em projetos Java

## Objetivo

Praticar a integração de aplicações Java com serviços externos, o tratamento adequado de erros e o armazenamento de informações em arquivos, desenvolvendo soluções mais completas e próximas de cenários reais.

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- APIs REST
- Bibliotecas nativas do Java para HTTP e arquivos
